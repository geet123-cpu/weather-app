# Weather Forecast Application

A full-stack weather forecasting application built using React.js, Node.js, Express.js, and OpenWeather API.

## Features

* Search weather by city
* Current location weather
* 5-day weather forecast
* Weather icons
* Search history management
* CRUD operations
* Responsive UI

## Tech Stack

* React.js
* Node.js
* Express.js
* OpenWeather API
* Git & GitHub
